天际弹窗登录API：

<!-- 页面引用脚本： 注，基于jQuery，所以要放在jquery文件引用之后-->
<script type="text/javascript" src="http://image.tianji.com/tjs/popup-login/js/tj-popup-login.min.js"></script>
<!-- 基本状态：默认为登录、保持登录状态为默认选中 -->
<a href="http://www.tianji.com/p" class="popup-login">基本状态：默认为登录、保持登录状态为默认选中</a>
<!--默认为登录、保持登录状态为默认不选中，成功跳转链接为job.tianji.com -->    
<a href="http://job.tianji.com/" class="popup-login" data-checked="false" data-target="http://job.tianji.com/">登录，默认不选中登录状态，成功跳转链接为job.tianji.com</a>
<!--默认为注册、保持登录状态为默认选中，注册成功后跳转至首页 -->    
<a href="http://www.tianji.com/home" class="popup-login" data-init="register" data-target="http://www.tianji.com/home">默认为注册、保持登录状态为默认选中，注册成功后跳转至首页</a>
<!--默认为注册、保持登录状态为默认不选中 -->    
<a href="javascript:void(0);" class="popup-login" data-init="register" data-checked="false">默认为注册、保持登录状态为默认不选中</a>
//可选属性列表：
{
    "data-checked": true,   //保持登录状态checkbox是否默认勾选，可选值：true|false，默认值：true
    "data-init": "login",   //点击打开弹窗登录默认显示登录或者注册，可选值："login"|"register"，默认值："login"
    "data-target": ""       //注册或登录后的跳转链接，默认为跳转回当前页，可指定任意URL，请以http开头
}

普通链接添加事件监听：

$('#test1').click(function (){
    var $this = $(this);
    $.ajax('aaa.html',{
        type: 'get',
        statusCode: {
            404: function (){
                popupFrameCreate($this.attr('data-target'));
            },
            500: function (){
                popupFrameCreate($this.attr('data-target'));
            }
        }
    });
    });

运用实例：

  • ·基本状态：默认为登录、保持登录状态为默认选中
  • ·登录，默认不选中登录状态，成功跳转链接为job.tianji.com
  • ·登录，默认不选中登录状态，成功跳转链接为/a/b.html
  • ·默认为注册、保持登录状态为默认选中，注册成功后跳转至http://www.tianji.com/mba
  • ·默认为注册、保持登录状态为默认不选中，已登录时点击刷新当前页
